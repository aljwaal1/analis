# منصة التحليل المالي V1.0

مشروع PWA عربي يعمل على Windows وAndroid وiPhone من المتصفح.

## التشغيل السريع على Windows
1. فك ضغط الملف.
2. شغّل الملف `تشغيل-محلي.bat`.
3. سيفتح التطبيق على `http://localhost:8080`.

يمكن فتح `index.html` مباشرة للتجربة، لكن التثبيت والعمل الكامل دون اتصال يحتاج تشغيله عبر خادم محلي أو GitHub Pages.

## الوظائف
- استيراد XLSX وXLS وCSV.
- رفع عدة ملفات والاختيار بين الاستبدال والإضافة.
- تعيين ذكي للأعمدة مع معاينة أول 8 صفوف.
- لوحة مؤشرات تنفيذية.
- التحليل الأفقي والرأسي.
- 24 مؤشرًا ونسبة مالية مع تفسير.
- تحليل المصروفات والإيرادات حسب البند والفئة والفرع ومركز التكلفة والعميل.
- مقارنة الجداول حسب الأبعاد.
- تحليل التدفقات النقدية والتدفق الحر.
- رسوم SVG تعمل محليًا.
- تصدير Excel وCSV وJSON وتقرير للطباعة أو الحفظ PDF.
- حفظ الجلسة محليًا عبر IndexedDB.

## بنية البيانات المدعومة
الفترة، القائمة المالية، الفئة، البند، القيمة، الفرع، مركز التكلفة، العميل، نوع التدفق النقدي، الموازنة، العملة.

## النشر على GitHub Pages
ارفع محتويات المجلد إلى مستودع GitHub ثم فعّل Pages من إعدادات المستودع. لا يحتاج المشروع إلى Build.
