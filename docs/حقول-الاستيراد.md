# حقول الاستيراد

الحقول الإلزامية: الفترة، البند، القيمة.

الحقول الاختيارية: القائمة المالية، الفئة، الفرع، مركز التكلفة، العميل، نوع التدفق النقدي، الموازنة، العملة.

يفضل استخدام أسماء البنود القياسية مثل: إجمالي الأصول المتداولة، المخزون، النقدية، إجمالي الخصوم المتداولة، إجمالي الأصول، إجمالي الخصوم، حقوق الملكية، إجمالي الإيرادات، تكلفة المبيعات، الربح التشغيلي، صافي الربح، مصروف الفوائد، صافي التدفق التشغيلي، شراء أصول.
