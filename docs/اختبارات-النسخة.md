# اختبارات النسخة V1.0

تم تنفيذ الفحوصات التالية قبل التسليم:

- فحص صياغة جميع ملفات JavaScript بواسطة Node.js.
- اختبار محرك التحليل للتأكد من عدم تكرار إجمالي الإيرادات مع تفاصيلها.
- اختبار حساب المصروفات وصافي الربح.
- اختبار التدفقات التشغيلية والاستثمارية والتمويلية والتدفق الحر.
- اختبار نسبة التداول ضمن محرك النسب المالية.
- التحقق من وجود جميع عناصر الواجهة المشار إليها من JavaScript.
- التحقق من وجود جميع الملفات المسجلة في Service Worker.
- التحقق من الروابط المحلية والقوالب وملفات العينات.
- فحص ملفات Excel والتأكد من سلامة الجداول وعدم وجود أخطاء صيغ.

ملاحظة: اختبار Chromium الآلي داخل بيئة الإنشاء مُنع بسياسة البيئة، لذلك تم الاعتماد على اختبارات المحرك والفحص الساكن الشامل. المشروع مصمم للتشغيل عبر ملف تشغيل Windows أو أي خادم محلي أو GitHub Pages.
